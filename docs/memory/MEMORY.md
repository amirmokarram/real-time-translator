# Real-Time Translator — Memory Index

Desktop app: real-time translation of system audio (meetings/video). **Language pair is user-configurable** (Settings → Languages; added 2026-06-26) — defaults to English→Persian, still the primary use case. Core pipeline (capture → DeepGram STT → switchable translation → live display) + overlay + export all working (built 2026-06-04). **Assist mode** (LLM Q&A about the conversation, 4 providers incl. local Ollama/Docker Model Runner) added 2026-06-07, merged to `main`. **Published 2026-06-07** as public GitHub repo https://github.com/amirmokarram/real-time-translator — **All Rights Reserved / proprietary** (LICENSE + `UNLICENSED` in package.json), README rewritten with full docs.
